== Description ==

Provides rich snippet rating/voting functionality for every content of a page.

You can customize size, orientation and colors of the rating/voting.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire `star-snippets` folder to the `/wp-content/plugins/` directory
1. DO NOT change the name of the `star-snippets` folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to `Settings > Star snippets` customize and save them.
1. Place your snippet e.g.`{star-snippet}` in your content (posts, pages)
